# SKALA Front-End 개인 포털

HTML과 CSS를 활용해 제작한 개인 포털 웹사이트입니다.  
프로필, 포트폴리오, 강의 시간표, 휴일 일과, 여행 앨범, 회원가입 페이지로 구성했습니다.

## 주요 페이지

- `index.html` : 메인 포털 페이지
- `myProfile.html` : 개인 소개 및 성장 과정
- `portfolio.html` : 프로젝트, 기술 스택, 수상 내역을 소개하는 포트폴리오
- `myClass.html` : 주간 강의 시간표
- `myHoliday.html` : 휴일 시간대별 일과
- `myTrip.html` : 여행 사진, 음악, 영상
- `signUp.html` : 회원가입 폼
- `signUpResult.html` : 회원가입 완료 페이지

## 사용 기술

- HTML5
- CSS3
- Git
- GitHub
- GitHub Pages

## 주요 기능

- 여러 HTML 페이지 간 이동
- 개인 프로필 및 성장 과정 제공
- 프로젝트와 기술 스택을 정리한 포트폴리오 제공
- 공통 CSS를 활용한 통일된 디자인
- Grid와 Flex를 활용한 레이아웃
- 모바일 화면을 고려한 반응형 디자인
- 이미지, 오디오, 비디오 콘텐츠 제공
- 표, 폼, 목록 등 다양한 HTML 요소 활용
- GitHub Pages를 이용한 정적 웹사이트 배포

## 포트폴리오 내용

포트폴리오 페이지에서는 다음 내용을 확인할 수 있습니다.

- 멀티에이전트 기반 면접 표현 습관 분석 시스템
- AI 기반 감귤 자동 수확 및 품질 분류 로봇
- TabPFN 기반 소프트웨어 결함 예측
- LSTM AutoEncoder 기반 시스템 이상 탐지
- 사용자 맞춤 레시피 추천 시스템
- Java 기반 Shoot the Duck 게임
- 프로그래밍 언어 및 기술 스택
- 수상 내역 및 자격증
- 향후 목표
